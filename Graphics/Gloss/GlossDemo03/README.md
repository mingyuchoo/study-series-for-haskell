# GlossDemo03

Gloss 라이브러리를 사용한 인터랙티브 게임 예제.

## 프로젝트 구조

```
GlossDemo03/
├── app/
│   └── Main.hs      -- 진입점
├── src/
│   └── Lib.hs       -- 게임 로직
├── test/
│   └── Spec.hs
├── package.yaml
└── stack.yaml
```

## 주요 기능

- **인터랙티브 그래픽스**: `play` 함수를 사용한 게임 루프
- **키보드 입력 처리**: 방향키로 원을 이동 (상/하/좌/우 각 10px)
- **게임 상태 관리**: `(x, y)` 튜플로 위치 추적
- **렌더링**: 현재 위치에 반지름 20의 원 표시

## 설치 및 실행 방법

```bash
# 빌드
stack build

# 실행
stack run
```

## 주요 의존성

| 패키지 | 용도 |
|--------|------|
| base | 기본 라이브러리 |
| gloss | 2D 그래픽스 라이브러리 |
